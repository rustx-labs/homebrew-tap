# Rustx-labs Eye-sentry

## How do I install these formulae?

`brew install rustx-labs/eye-sentry/<formula>`

Or `brew tap rustx-labs/eye-sentry` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rustx-labs/eye-sentry"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
